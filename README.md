## About

This is my **first university project**.

A simple **Blackjack console game written in pure C**.
It was created mainly as a learning exercise, so some parts of the code are **hardcoded or simplified**.

## Requirements

On **Windows**, you may need to enable **UTF-8 support** for the console to display characters correctly.

Go to:

`Control Panel → Region → Administrative → Change system locale`

Then enable:

`Use Unicode UTF-8 for worldwide language support`
